# Crunch Calculator

Crunch is a high-performance, professional-grade engineering calculator designed for the modern web. It combines raw power with a tactile, hardware-inspired user interface (Lux UX) to provide a mission control experience for mathematical exploration.

## Key Features

### 1. Natural Logic Engine
Powered by Math.js, Crunch understands complex mathematical expressions, units, and algebraic assignments. 
- **Assignments**: Type x = 50 * sin(45 deg) to save values.
- **Unit Conversion**: Easily convert between units like 50 km to miles.
- **Intelligent Hints**: Syntax errors stay out of your way until you compute, providing precise feedback when needed.

### 2. Numerical Fluidity
Never lose sight of your data. Crunch automatically scales its typography to prevent overflow, ensuring that even high-precision constants and massive engineering formulas remain fully visible and legible.

### 3. Visual Bitfield (Programmer Mode)
A dedicated 64-bit visual interactive bitfield for low-level logic. Toggle bits, view Hex/Dec/Bin representations in real-time, and sync them directly back into your main calculation stream.

### 4. Global Constants & Variables
- **Physics Constants**: Built-in access to speed of light, planck's constant, etc.
- **Dynamic Variable Inspector**: Manage your saved states with a swipeable variables panel.
- **Currency Sync**: Real-time exchange rate integration via external API.

### 5. The Lux UX Layer
- **Tactile Physics**: Physical button depth and haptic feedback (vibration) make every interaction feel precise.
- **Dynamic Lighting**: A mouse-reactive radial gradient adds depth to the hardware casing.
- **Gestural Control**: Swipe-to-dismiss overlays for a fluid, natural mobile experience.
- **Tabular Alignment**: Numbers stay locked in a fixed-width grid for rock-solid visual precision.

## Tech Stack

- **Framework**: React 18 + TypeScript
- **Styling**: Tailwind CSS
- **Math Engine**: Math.js
- **Animation**: Motion (Framer Motion)
- **Icons**: Lucide React

## Getting Started

1. **Calculate**: Simply type your formula into the f bar. 
2. **Crunch**: Press the orange Zap button or tap '=' on the keyboard.
3. **Reuse**: Tap any result in the Session Stream to pull it back into your current formula.
4. **Variables**: Assign a value (e.g., price = 200) to see it appear in the Variable Overlay.
